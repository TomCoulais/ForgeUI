## 🧭 Overview

ForgeUI is a powerful developer tool designed to streamline the creation and management of customizable design systems using Vue.js and Vite.

## 🚀 Getting Started

### 🛠️ Prerequisites

Make sure you have the following installed:

- Node.js >= 18
- npm or yarn
- Git

### 🧩 Installation

```bash
git clone https://github.com/TomCoulais/ForgeUI.git
cd forgeui
npm install
```
